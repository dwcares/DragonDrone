# DragonDrone
A Microsoft Cognitive Services computer vision demo built for a DJI Mavic Pro Drone

- [ ] P1: Facial verification

- [ ] P1: FPV view of done
- [ ] P1: showing successful detection
- [ ] P1: show in the app
- [ ] P2: Turn lights on/off when white walker found
